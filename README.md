## Tecnologias Utilizadas

- **Next.js**
- **TypeScript**
- **CSS Modules**

## Como Executar

1. Clone o repositório:

git clone https://github.com/VictorFDiniz/Nextjs-montyhall-project

2. Instale as dependências:

npm install

3. Inicie o servidor de desenvolvimento:

npm run dev
